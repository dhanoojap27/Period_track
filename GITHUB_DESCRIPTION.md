## Short Description (160 characters max)
🌸 Privacy-focused period tracker with ML-powered predictions, offline-first architecture, and optional cloud sync. Built with Flutter & Firebase.

## About Section

A comprehensive menstruation tracking application that puts privacy first. Track your cycle, predict future periods with machine learning algorithms, and gain insights into your reproductive health—all with an offline-first architecture and optional cloud synchronization.

### Key Highlights:
- 📊 **Smart ML Predictions**: Weighted moving average + linear regression for accurate cycle forecasting
- 🔒 **Privacy-Focused**: Offline-first with local Hive database, cloud sync is optional
- 📈 **Comprehensive Analytics**: Track symptoms, mood, flow levels, and view detailed insights
- 🎯 **Accurate Forecasting**: Confidence intervals, trend detection, and outlier filtering
- 🔔 **Smart Notifications**: Period reminders with customizable timing
- 🌐 **Cross-Platform**: Built with Flutter for iOS, Android, Web, and Desktop

### Tech Stack:
Flutter • Dart • Riverpod • Hive • Firebase • ML Algorithms

### Features:
✨ Period logging with symptoms and mood tracking
✨ ML-powered cycle predictions with confidence intervals
✨ Fertile window and ovulation day calculations
✨ Interactive charts and analytics
✨ Calendar view with color-coded indicators
✨ Offline-first with optional Firebase sync
✨ Local notifications and reminders
✨ No ads, no data selling
